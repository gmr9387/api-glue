### Contribution Guide

**1. Fork the repository:**
   - Click the "Fork" button at the top right corner of this repository to create your own copy.

**2. Clone your fork:**
```bash
   git clone https://github.com/<your-username>/api-glue.git
```

**3. Create a new branch:**
```bash
   git checkout -b feature/your-feature-name
```

**4. Implement your changes and commit:**
   - Follow best practices by keeping commits focused on specific functionalities.
```bash
   git commit -m 'Add feature: your feature name'
```

**5. Push to your forked repository:**
```bash
   git push origin feature/your-feature-name
```

**6. Create a Pull Request:**
   - Navigate to the original repository on GitHub and click **New Pull Request**.
   - Ensure your Pull Request clearly describes the changes you made and reference any issue related to it.

**7. Wait for the review and join discussions if necessary.**

**8. Done! 🎉 Your pull request will be merged once approved.**
